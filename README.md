# Kanban Board

A drag-and-drop Kanban board for visual project management, built with React and TypeScript.

## Features
- Four columns: Backlog, To Do, In Progress, Done
- Drag and drop cards between columns
- Add cards with title, description, and priority (Low / Medium / High)
- Delete cards on hover
- Persists to localStorage — survives page refresh
- Dark theme

## Tech Stack
React 18 · TypeScript · Tailwind CSS v4 · Vite · Lucide React

## Getting Started
```bash
npm install
npm run dev
```
